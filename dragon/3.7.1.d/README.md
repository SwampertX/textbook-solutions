# NFA to DFA converter

Input NFA, get the transition table for the DFA,
with DFA state to NFA states mapping.

Requirements:
For pretty printing.

```bash
pip3 install tabulate
```

Usage:

```bash
python3 main.py
# toy examples
python3 main.py < nfa_inputs/3.26.txt
python3 main.py < nfa_inputs/3.29.txt
```
